# URL Shortener (Flask)

Простой сервис сокращения ссылок, реализованный на Flask.

## Функционал

- Создание коротких ссылок
- Редирект на оригинальный URL
- Подсчет количества переходов
- Страница статистики

## Технологии

- Python
- Flask
- SQLite
- HTML/CSS
